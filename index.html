How does internet work

Just like people talking to each other using a particular language, computers talk to each other using their language. The most popular language is HTML. HTML stands for hyper text markup language. When I want to visit google, i enter the web address www.google.com into my browser Now my browser will send an HTTP request to the service provider and the service provider will find the server that holds the information The server then sends back a HTML document to the browser The browser will now interprets the data and displays on my computer. in this case when i wanted to look for google, google servers provides me with the informations.

Tags and eliments

HTML contents consist of HTML elements, and these elements consist of other tags and attributes.
this is how the browser know s what to do with the HTML document. Using all these together the computer could display any content.

What is Void tag

Void tags are basically the tags that does'nt have any content in it. So when using void tags we dont have to specifically mark the beginning and the end of those tags.

Inline or Block elements

HTML contents are either inline or block elements. this is used to mark the content inside the elements
